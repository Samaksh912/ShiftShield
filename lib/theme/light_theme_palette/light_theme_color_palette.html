# Light Theme Color Palette: Ignite Delivery System This palette is designed for high legibility in outdoor environments, maintaining the "Precision Velocity" aesthetic while providing a cleaner, daylight-optimized experience. ## 1. Core Palette | Token | Color | Hex Code | Usage | | :--- | :--- | :--- | :--- | | **Surface** | Soft Cloud Gray | `#F5F5F5` | Main screen background. | | **Primary** | Kinetic Orange | `#FF6A00` | CTAs, active states, and branding. | | **On-Surface (High)** | Jet Black | `#0E0E0E` | Main headings and primary text. | | **On-Surface (Med)** | Steel Gray | `#4A4A4A` | Sub-labels and secondary text. | ## 2. Elevation & Surfaces - **Surface Container:** `#FFFFFF` (Pure White) - Used for cards and input fields to pop against the gray background. - **Surface Container Low:** `#EEEEEE` - Used for subtle grouping and section dividers. ## 3. Accents & Feedback - **Success:** `#00C853` - Payout confirmations, claim approvals. - **Alert:** `#FF3D00` - Critical errors or coverage warnings. - **Info:** `#2979FF` - Helpful tips or rider support links. ## 4. Visual Strategy - **Contrast:** Maintains a high contrast ratio between Kinetic Orange and Soft Cloud Gray for outdoor visibility. - **Depth:** Uses soft, multi-layered shadows (`rgba(0, 0, 0, 0.05)`) to create a premium, tactile feel. - **Consistency:** All typography (Space Grotesk) and corner radii (8px) remain identical to the dark mode for brand continuity.
